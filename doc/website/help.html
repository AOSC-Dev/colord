How can I help?

Anyone interested in colord development is invited to join the channel #gnome-color-manager on freenode. We are a friendly bunch, and can answer questions on DBus interfaces, general lcms questions or anything else colord related.

There is also a subscription-only mailing list where we discuss colord development, patches and future development.

You can file bugs, or track the status at bugs.freedesktop.org

If colord cannot do what you need then please ask on the mailing list. It might already be on the TODO, or we might have to slightly tweak the API to make it possible.
